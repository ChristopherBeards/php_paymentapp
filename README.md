# PHP Payment App 
Built with PHP, MySQL, and Stripe.

# Quick Start

- Install dependencies
`composer install`

- Import paypage.sql into your DB

- Add your DB params in `config/db.php`

- Add your stripe pk and sk keys into `charge.js` and `charge.php`

- Visit server location
